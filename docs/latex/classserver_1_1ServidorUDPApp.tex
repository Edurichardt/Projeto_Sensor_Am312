\doxysubsection{Detailed Description}
Classe principal da aplicação Tkinter para o servidor UDP. 

Esta classe cria a janela gráfica, inicializa o servidor UDP em uma thread separada, e atualiza continuamente a interface conforme os dados recebidos.
